# Brainstorm - RDC Viagens

## Contexto
Recriação do site RDC Viagens com foco na página de assinaturas. O site original já existe e precisa manter a identidade visual estabelecida.

## Abordagem 1: Tropical Elegance

**Design Movement**: Luxury Resort Minimalism com influências de design brasileiro contemporâneo

**Core Principles**:
- Espaços amplos e respiráveis que evocam a sensação de férias
- Uso estratégico de imagens de destinos como elementos principais
- Tipografia elegante e legível que transmite confiança
- Transições suaves que simulam o movimento das ondas

**Color Philosophy**: 
- Azul profundo (#1E40AF) como cor principal - representa confiança, mar e céu
- Magenta vibrante (#E91E8C) como accent - energia e paixão por viajar
- Tons neutros quentes para backgrounds - acolhimento e sofisticação
- Gradientes sutis do azul ao magenta para CTAs

**Layout Paradigm**:
- Hero sections com imagens full-width e overlays sutis
- Cards de planos com bordas arredondadas e sombras suaves
- Seções alternadas com fundos claros e escuros
- Grid assimétrico para quebrar monotonia

**Signature Elements**:
- Ondas SVG como divisores de seção
- Ícones de linha fina com animação de hover
- Badges flutuantes com informações de destaque

**Interaction Philosophy**:
- Hover states que revelam mais informações
- Scroll animations suaves e não intrusivas
- Botão flutuante com pulse animation sutil

**Animation**:
- Fade-in on scroll para elementos
- Scale up suave em cards ao hover
- Pulse animation no botão flutuante de compra
- Transições de 300ms para interações

**Typography System**:
- Montserrat para títulos (700, 800)
- Poppins para corpo de texto (400, 500, 600)
- Hierarquia clara: 48px/36px/24px/18px/16px
</text>

## Abordagem 2: Dynamic Journey

**Design Movement**: Motion Design com influências de storytelling visual

**Core Principles**:
- Narrativa visual que guia o usuário pela jornada de compra
- Elementos que parecem estar em movimento constante
- Formas orgânicas e curvas que evitam rigidez
- Interatividade que engaja e surpreende

**Color Philosophy**:
- Azul RDC (#0047AB) como base sólida - tradição e confiabilidade
- Rosa vibrante (#FF1493) para highlights - emoção e aventura
- Amarelo dourado para badges premium - exclusividade
- Backgrounds com gradientes diagonais sutis

**Layout Paradigm**:
- Seções com clip-path diagonal para criar dinamismo
- Cards de planos em formato de "ticket de viagem"
- Elementos que "flutuam" com parallax leve
- Assimetria proposital nas composições

**Signature Elements**:
- Formas de avião/nuvem como elementos decorativos
- Linhas curvas conectando seções (como rotas de voo)
- Depoimentos em formato de cartão postal

**Interaction Philosophy**:
- Micro-interações em cada elemento clicável
- Feedback visual imediato em todas as ações
- Animações que contam uma história

**Animation**:
- Elementos entrando em cena como se estivessem "pousando"
- Cards que "viram" ao hover revelando mais info
- Contador animado para estatísticas
- Botão flutuante com efeito de "respiração"

**Typography System**:
- Montserrat Bold para impacto nos títulos
- Poppins Regular/Medium para legibilidade
- Tamanhos generosos para mobile-first
</text>

## Abordagem 3: Premium Serenity

**Design Movement**: Scandinavian Minimalism adaptado ao contexto tropical

**Core Principles**:
- Menos é mais - cada elemento tem propósito claro
- Whitespace generoso que transmite calma e organização
- Foco absoluto na conversão com clareza de informação
- Elegância através da simplicidade

**Color Philosophy**:
- Azul marinho (#1E3A5F) para autoridade e profissionalismo
- Coral suave (#FF6B6B) como accent vibrante mas não agressivo
- Branco e cinza claro como canvas principal
- Toques de dourado para elementos premium

**Layout Paradigm**:
- Grid limpo e organizado
- Cards de planos lado a lado com destaque visual no recomendado
- Seções com padding generoso
- Hierarquia visual clara e previsível

**Signature Elements**:
- Linhas finas como separadores elegantes
- Ícones minimalistas e uniformes
- Selo de "Mais Popular" discreto mas efetivo

**Interaction Philosophy**:
- Transições suaves e previsíveis
- Estados de hover claros mas não exagerados
- Foco na usabilidade e acessibilidade

**Animation**:
- Fade simples para elementos on scroll
- Elevação sutil em cards ao hover
- Botão flutuante com sombra que aumenta ao scroll
- Transições de 200ms para resposta rápida

**Typography System**:
- Montserrat para toda a interface (consistência)
- Pesos variados para criar hierarquia (300-700)
- Line-height generoso para legibilidade
</text>

## Abordagem Selecionada: Tropical Elegance (Abordagem 1)

Esta abordagem foi escolhida por:
1. Mantém a identidade visual já estabelecida do site RDC Viagens
2. O uso de azul e magenta está alinhado com a marca
3. As ondas SVG e elementos orgânicos seguem a diretriz de evitar formas quadradas
4. A tipografia Montserrat/Poppins está de acordo com as diretrizes da marca
5. O foco em imagens de destinos como elementos principais está alinhado com as diretrizes de design
